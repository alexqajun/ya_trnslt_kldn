#### ТЕСТОВЫЕ ДАННЫЕ

#### ПРЕДУСЛОВИЯ _(в каждом тест-кейсе)_
1. Открыть браузер
2. Перейти по адресу https://ya.ru
3. В строке поиска ввести запрос _переводчик онлайн_
4. Нажать кнопку **Найти**
